  • when(_:_:otherwise:)

    Adds a new nested Context to the receiver. A predicate will be implicitly created that will evaluate to true when the current Environment is equal to the given value.

    For example, you might use this if your environment was an enum:

    ctx.when(.someState) { newCtx in ... }

    rather than:

    ctx.when(.init({ $0 == .someState })) { newCtx in ... }
